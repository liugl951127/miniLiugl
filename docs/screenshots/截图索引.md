## 截图规范

- 格式: PNG (透明度好, 适合 UI)
- 分辨率: 1920×1080 或 1440×900 (Retina 可放大 2x)
- 大小: 单张 ≤ 500KB (压缩工具: tinypng.com)
- 命名: 数字-描述.png (如 `01-login.png`)
- 标注: 重点区域用红框/箭头 (用 Snipaste 工具)
- 隐私: 隐藏真实用户/密码/API Key
