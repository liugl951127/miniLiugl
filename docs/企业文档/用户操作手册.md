## 第 2 章 AI 智能对话

### 2.1 进入 AI 对话

登录后, 默认进入"AI 助手"页面:

**截图占位 2-1: AI 对话主页**

```
+----------------------------------------------------------+
|  [☰] MiniMax  [搜索框]                  [🔔3] [👤admin] |
+----------------------------------------------------------+
|         |                                                |
| 侧边栏   |    AI 智能助手                                |
|         |    ┌────────────────────────────┐             |
|  💬 对话  |    │ 👋 你好, 我是 MiniMax AI  │             |
|  🛠 工具  |    │    有什么可以帮你?        │             |
|  📚 知识  |    └────────────────────────────┘             |
|  📊 看板  |                                                |
|  ⚙ 设置  |    [输入框: 问我任何问题...]    [发送]        |
|         |                                                |
+----------------------------------------------------------+
```

### 2.2 发起对话

**步骤**:
1. 在底部输入框输入问题
2. 点击"发送"或按 `Enter` 键
3. 等待 AI 响应 (右上角转圈)
4. 响应完成后可继续追问

**示例问题**:
- "帮我写一份周报"
- "Python 怎么读取 CSV"
- "分析一下本月销售数据"

### 2.3 切换 AI 模型

**步骤**:
1. 点击顶部"模型"下拉框
2. 选择目标模型 (内置 / 插件化)
3. 后续对话将使用新模型

**截图占位 2-2: 模型选择**
```
[模型 ▼] ┌────────────────────┐
          │ ● 内置 (推荐)      │
          │ ○ 插件: 视觉       │
          │ ○ 插件: 推理       │
          └────────────────────┘
```

### 2.4 对话历史

- 左侧侧边栏显示历史对话列表
- 点击任一历史项可恢复上下文
- 长按可删除/重命名
- 点击"新建对话"开启新会话

### 3.2 调用工具 (以 NL2SQL 为例)

**步骤**:
1. 在工具市场点击"NL2SQL"
2. 进入工具详情页
3. 在输入框输入自然语言
   - 例: "查询上个月销售额 TOP 10 的客户"
4. 选择目标数据源
5. 点击"执行"
6. 查看生成的 SQL + 执行结果

### 3.3 生成 PPT

**步骤**:
1. 工具市场 → 选 "PPT 生成"
2. 选择输入方式 (Markdown / JSON / 纯文本 / 自动生成)
3. 选择主题 (商务蓝/暗夜/自然绿/暖橙)
4. 输入大纲或主题
5. 点击"生成"
6. 下载 `.pptx` 文件

### 3.4 图像理解

**步骤**:
1. 工具市场 → 选 "图像理解"
2. 上传图片 (支持 jpg/png/webp)
3. 选择模型 (内置 / 插件)
4. 点击"分析"
5. 查看描述结果

## 第 4 章 个人中心

### 4.1 资料管理

**路径**: 右上角头像 → "个人中心"

可修改:
- 头像
- 昵称
- 邮箱
- 手机号
- 个人简介

### 4.2 密码修改

**步骤**:
1. 个人中心 → "安全设置"
2. 输入旧密码
3. 输入新密码 (≥ 8 位, 含大小写+数字+特殊字符)
4. 确认新密码
5. 点击"保存"

### 4.3 API Key 管理

**路径**: 个人中心 → "API 密钥"

每个用户可创建多个 API Key, 用于调用 OpenAI 兼容协议:
```
curl -X POST http://your-domain.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "minimax-ai", "messages": [...]}'
```

### 4.4 我的收藏

收藏的 AI 工具 / Agent / 模型在"个人中心 → 收藏"中查看.

## 第 6 章 常见问题 FAQ

### Q1: 忘记密码怎么办?

**答**:
1. 登录页点击"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件
4. 点击链接重置 (1 小时内有效)
5. 如未收到, 联系管理员重置

### Q2: AI 响应很慢?

**答**:
1. 检查网络连接
2. 切换到"内置"模型
3. 避开高峰时段
4. 联系管理员检查 GPU 资源

### Q3: 上传文件失败?

**答**:
1. 检查文件大小 (默认 ≤ 100MB)
2. 检查文件格式 (PDF/Word/Excel/图片)
3. 清除浏览器缓存重试
4. 切换浏览器 (Chrome 80+)

### Q4: 工具调用报错?

**答**:
1. 检查输入参数
2. 查看错误码 (工具详情页有说明)
3. 联系技术支持 (附上 traceId)

### Q5: 如何邀请同事?

**答**:
1. 管理员登录
2. 管理 → 用户管理 → 新建用户
3. 填写信息, 分配角色
4. 系统自动发送激活邮件

### Q6: 支持哪些浏览器?

**答**: Chrome 80+ / Edge 79+ / Firefox 70+ / Safari 13+ (推荐 Chrome)

### Q7: 支持移动端吗?

**答**: 支持
- 浏览器访问: 自动适配
- 原生 App: 见 [移动端开发文档](../MOBILE.md) (iOS/Android via Capacitor)

### Q8: 数据是否安全?

**答**:
- 数据本地存储 (企业内网)
- 传输加密 (HTTPS / TLS 1.2+)
- 密码 BCrypt 哈希
- 审计日志全留痕
- 详见 [安全白皮书](#) (企业版)
