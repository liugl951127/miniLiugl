## 二、🔴 高优先级 (P0 - 必须解决)

### 2.1 前端 AI 工具页面 SDK 不一致

**问题**: `frontend/src/views/ai/AiToolAdmin.vue` 用自建 `axios.create()`,未使用 `frontend/src/api/ai.js` 统一 SDK

**影响**:
- 升级/维护成本高
- 与新后端接口 `/api/ai/admin/codegen` 等不兼容
- 无 traceId / 错误统一处理

**修复方案**:
```js
// 当前 (问题)
const api = axios.create({ baseURL: '/api/ai/admin', timeout: 60000 })

// 改为
import { listTools, createTool, invokeTool, listDataSources } from '@/api/ai'
```

**状态**: ⏳ 待修复 (本审计发现,优先级最高)

| 问题 | 描述 | 建议 |

### 5.4 数据库设计

## 八、结论

3. 下季度: AI 工作流 + Agent 框架
